# Poloroid - A Social Media Website

Poloroid is a modern social media platform that allows users to connect, share moments, and interact in a safe and moderated environment. Users can sign up, log in, upload images with captions, send friend requests, and view posts from their network—all while benefiting from built-in AI-based content moderation.

---

## Features

- **User Authentication**  
  Secure user registration and login system.

- **Picture Upload with Captions**  
  Share moments by uploading images along with captions.

- **AI-Based Content Moderation**  
  Automatically filters abusive or inappropriate content (both text and images) at the time of post upload.

- **Friend Requests System**  
  Send, receive, and accept friend requests to build your network.

- **Personalized News Feed**  
  View posts from accepted friends in a customized feed.

---

## Technologies Used

- **React.js** – Frontend library for building interactive user interfaces  
- **Django** – Backend framework for handling APIs and business logic  
- **MongoDB** – NoSQL database for storing user data and posts  
- **Tailwind CSS** – Utility-first CSS framework for responsive UI design  
- **Cloudinary** – Image storage and delivery  
- **AI/NLP Models** – For detecting and filtering abusive content in captions and images
